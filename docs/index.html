This is a collection of experimental projects, prototypes, and proof-of-concepts. Each project lives in its own directory under projects/ with independent dependencies and build configurations, managed as npm workspaces.

Skynet AGI

AI AGI Neural Network Self-Modification

A functional artificial general intelligence system with self-modifying neural architecture. Features include metacognition, emotional systems, pattern recognition, reasoning chains, and experimental self-modification capabilities.

Implements multi-layer neural networks with Hebbian learning, attention mechanisms, memory consolidation, and autonomous goal-directed behavior.
